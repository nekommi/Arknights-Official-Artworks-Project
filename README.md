# Arknights Official Artworks Project

A web reader for the Arknights Official Artworks volumes.

The scans presented here were originally shared by fans on Chinese websites. This site is a reader port intended to make the material more accessible for users who do not read Chinese or do not have access to a Baidu account.
I believe the original scanner is 艺视, but there have been so many sources, I can't be sure.

If you are a rights holder and believe any content here infringes your rights, please contact me through GitHub and I will review it promptly.

If you enjoy this reader, a small tip helps cover Cloudflare Image Storage costs.
